# SmartStadium AI - Professional Venue Navigator

SmartStadium AI is a premium, high-fidelity venue navigation platform designed to optimize the attendee experience during large-scale events. By integrating real-time sensor data with intelligent persona-based routing, the app provides personalized, crowd-aware journeys for every guest.

![SmartStadium AI](index.html)

## 🚀 Key Features

- **Interactive Stadium Map**: A 5x5 grid-based navigation map with real-time status glows (Low, Medium, High crowd density) and interactive node tooltips.
- **Intelligence Personas**: Specialized routing logic for **Normal**, **Family (Accessible)**, and **VIP (Priority)** guests.
- **AI-Optimized Journey**: Calculates the most efficient path based on live stadium bottlenecks, including a premium 1.5s "thinking" animation with background blurring.
- **Comprehensive Auth System**: Includes a secure authentication overlay with email, password, and Google Sign-in simulation.
- **Emergency Code Red**: A one-tap emergency mode that activates prioritized exit paths and overrides standard navigation.
- **Live Dashboard**: Real-time monitoring of stadium capacity, average wait times, and recommended gate entries.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Modern CSS3 with CSS Variables, Flexbox/Grid, Glassmorphism, and custom SVG integration.
- **Typography**: Space Grotesk (Tech/Headers) and Inter (UI/Body).
- **Icons**: Custom Feather-inspired SVG icons for a consistent, premium aesthetic.

## 📱 How to Use

1. **Sign In**: Access the stadium's dashboard using the authentication overlay.
2. **Select Persona**: Choose your attendee profile (e.g., VIP for the fastest routes).
3. **Pick a Goal**: Whether it's finding your seat, the nearest washroom, or the best food court.
4. **Optimize**: Hit "Optimize Journey" to let the Navigator AI calculate your best path.
5. **Navigate**: Use the highlighted nodes and tooltips on the interactive map to reach your destination.
